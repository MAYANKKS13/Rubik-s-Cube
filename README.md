# Rubik-s-Cube



The different methods by which we can solve a Rubik's Cube are:-
<UL><LI>Beginner Method.</LI>
<LI>CFOF Method</LI>
<LI>Roux Method</LI>
<LI>Z Method</LI>
<LI>Old Pochmann Method (For Blindfolded Solving)</LI></UL>

<H2>Beginner Method</h2>
In this method, Cube is solved layer by layer. First laye is solved by solving the edges or making a cross and then placing the corners. Then the 4 edge cubie is placed at right place end up solving the 2nd layer. At last, last layer is solved by making the cross, permutating the corners and then orienting the corners and up solving the cube.
